1. What is Semantic HTML?

19. What is the difference between a class and an id? - Give me an example of when I might use each one
	-Id's are more specific, applicable only to one element, whereas classes can be used multiple times. An example would be if you wanted to style a unique div, you'd use ID. If you want to style multiple, similar divs you would use a class for each one.
